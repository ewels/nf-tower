Welcome in Nextflow Tower

Nextflow Tower allows you to monitor the execution of any of your Nextflow data analysis pipelines.

1. Install Nextflow

If you don't have it already install Nextflow as usual: 

curl https://get.nextflow.io | bash

Finalise the installation copying the nextflow launcher in a directory in your $PATH.
